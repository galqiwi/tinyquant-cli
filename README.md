# TinyQuant CLI

A simple hello world CLI tool.

## Installation

```bash
uv pip install tinyquant-cli
```

## Usage

After installation, you can run the CLI with:

```bash
tinyquant-cli
```

This will output: `Hello, World from TinyQuant CLI!`

## Development

To install in development mode:

```bash
uv pip install -e .
```
